# PHP Study Project - 掲示板アプリ

## 📘 概要

このプロジェクトは **PHPのバックエンド開発学習** を目的とした個人制作の掲示板アプリです。
チーム開発で同様の掲示板を作成する予定があるため、
**全体像を理解し、実装フローを掴むための予習プロジェクト** として作成しています。（AIサポートでのインプット用）

---

## 🧩 現在の機能

* 🔐 **ログイン機能**
  既存ユーザーがIDとパスワードでログイン可能。

* 📝 **新規登録機能**
  新しいユーザーを登録し、登録直後にそのままログイン可能。

* 💾 **ユーザーデータ管理**
  `DBtext.txt` にユーザー名とパスワードを保存（簡易的なテキスト管理方式）。
   チャットの履歴は`post.txt`で保存

* 💬 **掲示板投稿機能**
  テキスト形式でメッセージを投稿可能。

* 👀 **投稿表示機能**
  投稿内容を一覧で表示し、**誰が投稿したのか**が分かる形式で出力。
